📖 User Guide

Getting Started

Prerequisites

  • Python 3.7+ (for simulation and reporting)
  • PowerShell 5.0+ (for running scripts)
  • Vivado (optional, for FPGA synthesis)

Installation

git clone <repository>
cd nrml_vlsi_project
# All tools are self-contained!

Running Simulations

Quick Start (PowerShell)

# Run all simulations
.\\scripts\\run_sim.ps1

# Run specific module
.\\scripts\\run_sim.ps1 alu
.\\scripts\\run_sim.ps1 mux
.\\scripts\\run_sim.ps1 counter

View Results

# Generate test report
python .\\scripts\\test_report.py

# View waveforms in terminal
python .\\scripts\\vcd_viewer.py wave\\alu_4bit.vcd
python .\\scripts\\vcd_viewer.py wave\\mux4to1.vcd
python .\\scripts\\vcd_viewer.py wave\\counter3.vcd

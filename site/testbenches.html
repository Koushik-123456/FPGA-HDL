🧪 Testbenches

ALU Testbench (alu_4bit_tb.v)

Comprehensive testbench for the 4-bit ALU module with 21 test vectors covering all operations.

Test Cases:

  • ADD Operations (4 tests): Basic addition, carry propagation, zero result, carry-in handling
  • SUB Operations (3 tests): Subtraction, underflow handling, edge cases
  • AND Operations (3 tests): All zeros, mask operations, reflexive
  • OR Operations (3 tests): Full ones, identity, zero input
  • XOR Operations (2 tests): Toggle, self-XOR (zero)
  • XNOR Operations (2 tests): Complement, reflexive
  • PASS A Operations (2 tests): Data passthrough, zero passthrough
  • NOR Operations (1 test): All zero input

Status: ✅ 21/21 PASSED

MUX Testbench (mux4to1_tb.v)

Testbench for the 4-to-1 multiplexer with comprehensive input and select combinations.

Test Cases:

  • Basic Selection (4 tests): sel=00,01,10,11 with data [1,2,3,4]
  • Pattern Testing (4 tests): sel=00,01,10,11 with data [F,A,5,0]
  • Edge Cases (2 tests): All zeros, all ones

Status: ✅ 10/10 PASSED

Counter Testbench (counter3_tb.v)

Testbench for the 3-bit synchronous counter covering reset, enable, and counting operations.

Test Phases:

  • Phase 1 - Reset Active (9 cycles): Verify reset holds counter at 0
  • Phase 2 - Counting (8 cycles): Verify increment from 0-7 with enable high
  • Phase 3 - Disabled (5 cycles): Verify counter holds value when enable is low
  • Phase 4 - Resume (3 cycles): Verify resume counting from held value

Status: ✅ ALL PHASES PASSED
